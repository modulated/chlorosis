# fungbc
GBC emulator

## Memory Map
- 8 KB Work RAM
- Cartrige space
- Video RAM
- IO map
- Interrupt handlers

## CPU
- Sharp SM83
- Z80 based (without IX or IY registers)
- 4.19 MHz
- Memory mapped peripherals

## Video
- 160 x144 pixels
- GB has 4 shades (white, light grey, dark grey, black)
- Color has 16 shade palette
